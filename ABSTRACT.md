# Abstract

**Gut Guardian: A Gamified Mobile Application for Personalized Gut Health Tracking**

Gut health tracking is essential for identifying food sensitivities and dietary patterns that impact individual wellbeing. However, maintaining consistent food and symptom logs over extended periods presents significant adherence challenges. Current health tracking applications often lack engaging mechanisms to motivate users through the repetitive daily logging process, resulting in incomplete data and unreliable pattern identification.

Gut Guardian addresses this challenge by transforming the 14-day gut health investigation into an interactive detective mystery game. Users assume the role of a health detective solving their personal gut health case, where each meal log becomes a piece of evidence and daily symptom tracking reveals clues to identify problematic foods. The application employs a comprehensive gamification strategy incorporating narrative storytelling, progression mechanics, visual feedback, and achievement systems to sustain user engagement throughout the tracking period.

The application implements multiple gamification elements aligned with established behavior change frameworks. An experience point (XP) and leveling system rewards consistent logging behavior, while a visual 14-day calendar provides transparent progress tracking with color-coded completion indicators. An animated guide character delivers contextual onboarding and reinforces the detective narrative throughout the user journey. The Evidence Board feature visualizes correlations between consumed foods and reported symptoms, transforming raw data into actionable insights presented within the detective theme. Achievement badges and milestone rewards celebrate user progress, while mission-based daily tasks provide clear objectives that reduce cognitive load.

Built with React Native and Expo, Gut Guardian delivers a cross-platform experience optimized for mobile devices while maintaining web accessibility. The application employs a warm, approachable color palette featuring mint green gradients and golden accents, deliberately designed to evoke friendliness and reduce the clinical associations often present in health applications. Smooth animations and micro-interactions enhance perceived responsiveness and create an engaging user experience that encourages continued interaction.

The technical implementation prioritizes user privacy through local-first data storage, eliminating server dependencies and external data transmission. The responsive design adapts seamlessly across iOS, Android, and web platforms, ensuring accessibility regardless of device preference. Custom animation systems provide fluid transitions that reinforce game-like qualities while maintaining functional clarity.

Gut Guardian demonstrates how gamification principles can be strategically applied to health behavior tracking, transforming a mundane self-monitoring task into an engaging experience. By framing gut health investigation as a solvable mystery and providing continuous feedback through game mechanics, the application addresses the fundamental adherence challenge in long-term health tracking. Future enhancements could incorporate machine learning for automated pattern detection, social features for community support, and integration with wearable devices for comprehensive health data collection.
